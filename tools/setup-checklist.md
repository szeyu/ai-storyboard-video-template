# Setup Checklist

## New Project Setup

- [ ] Create a new repo from this template
- [ ] Open the repo in your AI coding tool
- [ ] Ask the AI to read `AGENTS.md`
- [ ] Fill in `00-brief/project-brief.md`
- [ ] Add source references into `00-brief/references/`
- [ ] Add source assets into `04-assets/`
- [ ] Ask the AI to generate ideas
- [ ] Select one idea
- [ ] Generate storyboard
- [ ] Create scene folders from storyboard automation
- [ ] Generate prompts for each scene
- [ ] Generate first-frame candidates for each scene
- [ ] Generate last-frame candidates when useful
- [ ] Generate scene video attempts
- [ ] Point timeline to the accepted attempt for each scene
- [ ] Build timeline
- [ ] Assemble final video
- [ ] Create thumbnail
- [ ] Create captions
- [ ] Create posting copy
- [ ] Prepare client handoff
