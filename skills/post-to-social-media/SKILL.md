# Post to Social Media

## Purpose

Use this skill when the final video is ready and the user wants to prepare or publish it to social platforms.

## Current Status

This skill is a placeholder.

For now, prepare posting assets only:

- final video
- caption
- hashtags
- thumbnail
- platform-specific copy

## Supported Platforms

Potential platforms:

- TikTok
- Instagram Reels
- YouTube Shorts
- LinkedIn
- Facebook

## Rules

- Never publish without explicit user approval.
- Always confirm platform, account, caption, and final file.
- Keep posting copy under `05-final/posting-copy/`.
